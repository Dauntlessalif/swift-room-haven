## Navigation Flow

```
Landing Page (/)
    │
    ├── Customer Login (/customer)
    │      │
    │      ├── Enter Email
    │      └── Sign In
    │           │
    │           ├── My Bookings Tab
    │           │      ├── View Upcoming
    │           │      ├── View Past
    │           │      ├── Cancel Booking
    │           │      └── Download Invoice
    │           │
    │           ├── Profile Tab
    │           │      ├── View Info
    │           │      ├── Edit Mode
    │           │      └── Save Changes
    │           │
    │           ├── Reviews Tab
    │           │      ├── View Reviews
    │           │      ├── Edit Review
    │           │      └── Delete Review
    │           │
    │           ├── Pet Care Tab
    │           │      ├── View Requests
    │           │      ├── New Request
    │           │      └── Cancel Request
    │           │
    │           └── Favorites Tab
    │                  ├── View Favorites
    │                  ├── Quick Book
    │                  └── Remove Favorite
    │
    └── [Logout] → Returns to Login
```
