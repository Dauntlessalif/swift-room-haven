## 🎨 Status Color Guide (Quick Reference)

### Bookings:
- 🟡 Yellow = Pending (action needed)
- 🟢 Green = Confirmed
- 🔵 Blue = Checked In
- ⚫ Gray = Checked Out
- 🔴 Red = Cancelled

### Messages:
- 🔵 Blue = New (unread)
- ⚫ Gray = Read
- 🟢 Green = Replied
- 🟣 Purple = Archived

### Reviews:
- 🟡 Yellow = Pending moderation
- 🟢 Green = Approved
- 🔴 Red = Rejected
